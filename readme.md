# Go Reloaded

Καλωσόρισες στο **Go Reloaded**! 🚀

Αυτό το project είναι μια απλή άσκηση σε Go, ιδανική για αρχάριους που θέλουν να γνωρίσουν τη γλώσσα και να εξασκηθούν σε βασικές λειτουργίες επεξεργασίας αρχείων και strings.

---

## 📌 Τι κάνει το πρόγραμμα

Το πρόγραμμα διαβάζει ένα αρχείο εισόδου (input file) και εφαρμόζει συγκεκριμένους κανόνες μετατροπής στο κείμενο.

Εκτός από τις μετατροπές λέξεων, το πρόγραμμα φροντίζει και για τη σωστή τοποθέτηση της στίξης, εξασφαλίζοντας ότι σημεία όπως τελείες, κόμματα και θαυμαστικά είναι κολλημένα στη λέξη που προηγείται και έχουν ένα κενό μετά. Ειδικές περιπτώσεις όπως "..." ή "!?" αναγνωρίζονται ως ενιαία σύνολα και δεν χωρίζονται. Επίσης, χειρίζεται σωστά τα εισαγωγικά τύπου ‘ ’, τοποθετώντας τα χωρίς κενά γύρω από το περιεχόμενο που περικλείουν.

Τέλος, εφαρμόζεται και ένας γραμματικός κανόνας: όταν η λέξη "a" προηγείται από λέξη που ξεκινά με φωνήεν ή με "h", αντικαθίσταται αυτόματα από "an", ώστε το κείμενο να είναι σωστό συντακτικά. Έπειτα γράφει το αποτέλεσμα σε ένα νέο αρχείο.

Κατάλληλο για:

* Αρχάριους που μαθαίνουν Go
* Εξάσκηση σε λειτουργίες αρχείων (file handling)
* Κατανόηση βασικών string operations και λογικής προγραμματισμού
